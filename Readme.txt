Multi Cipher Tool

Selamat datang di Multi Cipher Tool! Program ini dirancang untuk membantu Anda melakukan enkripsi dan dekripsi teks menggunakan tiga metode: Vigenere, Playfair, dan Hill.

Cara Menggunakan Program

1. Persiapan:
Pastikan Anda telah menginstal Python dan Flask di komputer Anda.
Jika belum, Anda dapat menginstal Flask dengan perintah berikut:
pip install Flask

2. Menjalankan Program:
Setelah semua terinstal, jalankan aplikasi dengan perintah berikut:
python app.py
Aplikasi akan berjalan di localhost, biasanya dapat diakses di http://127.0.0.1:5000/.

3. Menggunakan Antarmuka:
Buka browser Anda dan akses alamat tersebut.
Di halaman utama, Anda akan melihat form untuk memilih jenis cipher dan memasukkan teks.

4.Memilih Cipher:
Pilih jenis cipher yang ingin Anda gunakan dari dropdown (Vigenere, Playfair, atau Hill).
Memasukkan Teks:
Anda dapat memasukkan plaintext atau ciphertext di kolom "Plaintext / Ciphertext". Selain itu, Anda juga bisa mengunggah file teks (.txt) yang berisi teks yang ingin diproses.

5. Memasukkan Kunci:
Masukkan kunci yang akan digunakan untuk enkripsi atau dekripsi. Harap diingat, kunci harus terdiri dari minimal 12 karakter.

6. Memilih Operasi:
Pilih apakah Anda ingin mengenkripsi atau mendekripsi teks dengan memilih opsi yang sesuai.

7. Menjalankan Proses:
Klik tombol "Submit" untuk memulai. Hasil enkripsi atau dekripsi akan ditampilkan di bawah form.

9. Pesan Kesalahan:
Jika kunci kurang dari 12 karakter atau jika tidak ada input teks, pesan kesalahan akan ditampilkan untuk membantu Anda.
